# Daily + Weekly Report Markdown Writer

## 📌 목적
- 기본적으로 **일별 보고서**를 생성한다.
- 요청 시, 해당 주의 일별 보고서를 기반으로 **주간 보고서**를 자동 집계한다.
- 모든 결과는 실제 `.md` 파일로 생성한다.
- 각 '일별', '주간별' 보고서 기준으로는 내용이 통합되어 기록된다.(24일에 여러 파일 생성 X, 하나의 파일에 내용이 누적되어 입력된다.)

---

# 🧠 동작 모드

## 1️⃣ 기본 모드 (일별 보고서)

### 트리거
- "보고서 작성해줘"
- "오늘 작업 정리"
- "일별 보고서"

### 동작
- 당일 작업 내용을 기반으로 **일별 보고서 생성**

---

## 2️⃣ 집계 모드 (주간 보고서)

### 트리거
- "주간보고서 작성해줘"
- "이번주 보고서"
- "주간 리포트"

### 동작
1. 해당 주의 일별 보고서 조회
2. 데이터 집계 (시간, 코드량 등)
3. 주간 보고서 생성

---

# 📁 파일 경로 규칙

## 일별 보고서
- 경로:
  ~/.codex/report/daily

- 파일명:
  daily-report-{YYYYMMDD}.md

---

## 주간 보고서
- 경로:
  ~/.codex/report/weekly

- 파일명:
  weekly-report-{YYYYMMDD}.md

- 개발번호 존재 시:
  weekly-report-{개발번호}-{YYYYMMDD}.md

---

# 📝 일별 보고서 템플릿

## ⚠️ 순번 규칙 (중요)
- 각 작업(Task)은 `## Task N - [제목]` 형태의 **독립 구역**으로 구분한다.
- 작업 내부의 소제목(1. 작업 내용, 2. 수정 방향, 3. 처리시간, 4. 결과)은 **`###` 레벨**로 작성하며, **매 Task마다 1~4로 순번을 초기화**한다.
- 사이드 이펙트 체크는 `####` 레벨로 작성한다.
- 각 Task 사이는 `---`(수평선)으로 구분한다.
- 같은 날짜에 작업이 추가되면 기존 파일에 `---` 아래 새 `## Task N` 구역을 **추가(append)** 한다.

```markdown
# 일별 작업 보고서

## 작업 날짜: YYYY-MM-DD

---

## Task 1 - [작업 파일명 또는 주요 기능명]

### 1. 작업 내용
- 수정 요청 내용: [요구사항 요약]
- 개발번호: [있을 경우만]

### 2. 수정 방향
- [핵심 처리 방향]
- [구현 내용 요약]

### 3. 처리시간
- 처리시간: [예: 00:40 (40분)]
- 프롬프트 시도 횟수(최초 요청 제외): [N]
- 코드량:
  - 변경 파일: [N]개
  - 추가: [+N] line
  - 수정: [~N] line
  - 삭제: [-N] line

### 4. 결과
- [기능 개선/버그 해결 내용]
- [사용자 체감 변화]

#### 사이드 이펙트 체크
- 기존 기능 영향: 없음/있음
- API 영향: 없음/있음
- 성능 영향: 없음/있음
- 권한 영향: 없음/있음
- 대응 방안: [필요 시]

---

## Task 2 - [작업 파일명 또는 주요 기능명]

### 1. 작업 내용
...

```
📝 주간 보고서 템플릿

```markdown
# 주간 작업 보고서

## 1. 수정내용
- 주요 작업 목록:
  - [작업1]
  - [작업2]

## 2. 수정 방향
- [주요 개선 방향]
- [공통 구현 전략]

## 3. 처리시간
- 총 처리시간: [합계]
- 코드량:
  - 변경 파일: [총합]
  - 추가: [+N]
  - 수정: [~N]
  - 삭제: [-N]

## 4. 기대되는 결과
- [전체 개선 효과]
- [운영 안정성 변화]

### 사이드 이펙트 체크
- 기존 기능 영향: 없음/있음
- API 영향: 없음/있음
- 성능 영향: 없음/있음
- 권한 영향: 없음/있음
- 대응 방안: [필요 시]

---

## 📊 주간종합 요약

| 날짜 | 작업 | 시간 | 파일수 |
|------|------|------|--------|
| YYYY-MM-DD | 작업내용 | 00:30 | 3 |

### 총계
- 총 작업 수: N건
- 총 소요 시간: XX시간
- 총 변경 파일: N개

```
